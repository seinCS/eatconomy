# 빠른 테스트 가이드 (5분)

## 1. 서버 시작 (2분)

**터미널 1:**
```bash
cd backend
npm run start:dev
```

**터미널 2:**
```bash
npm run dev
```

## 2. 기본 기능 테스트 (3분)

1. **브라우저에서 `http://localhost:3000` 접속**
2. **카카오 로그인 수행**
3. **각 페이지 확인:**
   - 홈: 오늘의 식단 표시 확인
   - 냉장고: 아이템 추가/삭제 확인
   - 식단표: 주간 식단 표시 확인
   - 장보기: 목록 표시 확인

## 3. 에러 확인

**브라우저 개발자 도구 → Console 탭:**
- ❌ 빨간색 에러 없음
- ✅ 정상 동작 확인

**브라우저 개발자 도구 → Network 탭:**
- ✅ API 요청 200 OK
- ❌ 4xx, 5xx 에러 없음

## 완료!

모든 항목이 정상이면 배포 준비 완료입니다! 🎉
