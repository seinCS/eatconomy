# 🎉 Eat-conomy 패치 노트

## v1.2.0 - 식단 추천 시스템 고도화 업데이트
**배포일**: 2024년 12월 10일

---

## 🌟 주요 변경사항

### 1. 메인+반찬 세트 구성으로 더 현실적인 식단 추천

이제 각 식사에 **메인음식 1개 + 반찬 1개**가 함께 추천됩니다!

**변경 전:**
- 각 식사에 메뉴 1개만 추천
- 실제 식단 구성에 어려움

**변경 후:**
- 메인음식과 반찬이 함께 추천
- 더 현실적이고 균형 잡힌 식단 구성

### 2. 스마트 재료 연결 알고리즘

**하루 내 재료 반복 최소화**
- 같은 날 점심과 저녁에 같은 재료가 반복되는 것을 줄였습니다
- 더 다양한 식단을 즐길 수 있습니다

**다른 날 재료 연결 우선**
- 이전 날 저녁과 오늘 점심에 재료가 연결되면 가산점 부여
- 냉장고 재료를 효율적으로 활용할 수 있습니다

### 3. 점심/저녁에 맞는 요리 추천

- **점심**: 가벼운 요리 우선 추천 (칼로리 낮은 메뉴)
- **저녁**: 든든한 요리 우선 추천 (고기류, 고칼로리 메뉴)

## ✨ 새로운 기능

### 이번주 식단 다시 추천받기

홈 화면에서 **"이번주 식단 다시 추천받기"** 버튼을 통해 언제든지 새로운 식단을 받을 수 있습니다.

**사용 방법:**
1. 홈 화면 하단의 "이번주 식단 다시 추천받기" 버튼 클릭
2. 스와이프 페이지로 이동하여 선호도 조사
3. 새로운 식단 자동 생성

**장점:**
- 취향이 바뀌었을 때 쉽게 재설정 가능
- 다양한 식단을 시도해볼 수 있음

## 🎨 UI/UX 개선

### 오늘의 식단 화면 개선

**변경 전:**
- 메인음식만 표시
- 반찬 정보 없음

**변경 후:**
- 메인음식과 반찬 모두 표시
- 반찬은 작은 카드로 구분하여 표시
- 더 완성도 높은 식단 정보 제공

### 냉장고 페이지 정리

사용자 피드백을 반영하여 사용률이 낮은 기능을 제거했습니다.

**제거된 기능:**
- "이 재료로 식단 추천받기" 버튼

**이유:**
- 테스터 사용률이 매우 낮음
- 홈 화면에서 식단 재추천이 더 직관적

## 💡 사용 팁

### 더 나은 식단 추천을 받으려면

1. **냉장고에 재료를 충분히 추가하세요**
   - 재료가 많을수록 더 다양한 식단 추천 가능

2. **스와이프로 선호도를 정확히 표시하세요**
   - 좋아하는 메뉴는 오른쪽으로 스와이프
   - 싫어하는 메뉴는 왼쪽으로 스와이프

3. **식단이 마음에 들지 않으면 다시 추천받으세요**
   - 홈 화면의 "이번주 식단 다시 추천받기" 버튼 활용
