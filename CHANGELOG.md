# 패치 노트 (Changelog)

## [v1.2.0] - 2024년 12월 10일

### 🎉 주요 업데이트

#### 식단 추천 시스템 고도화
- **메인+반찬 세트 구성**: 이제 각 식사에 메인음식 1개와 반찬 1개가 함께 추천됩니다
- **스마트 재료 연결 알고리즘**: 
  - 하루 내 재료 반복을 최소화하여 식단 다양성 향상
  - 다른 날 재료 연결을 우선시하여 냉장고 재료 활용도 극대화
- **점심/저녁 적합성**: 
  - 점심에는 가벼운 요리 우선 추천
  - 저녁에는 든든한 요리 우선 추천

### ✨ 새로운 기능

- **이번주 식단 다시 추천받기**: 홈 화면에서 언제든지 새로운 식단을 추천받을 수 있습니다
  - 스와이프 선호도 조사부터 다시 시작
  - 기존 선호도 초기화 후 새 식단 생성

### 🎨 UI/UX 개선

- **오늘의 식단 화면 개선**:
  - 메인음식과 반찬을 모두 표시
  - 반찬은 작은 카드로 구분하여 표시
  - 더 직관적인 식단 정보 제공

- **냉장고 페이지 정리**:
  - 사용률이 낮은 "이 재료로 추천받기" 버튼 제거
  - 재료 관리에 집중할 수 있도록 UI 간소화

### 🐛 버그 수정

- Swipe 페이지에서 `TOTAL_CARDS` 미정의 오류 수정
- CORS 설정 개선으로 프로덕션 환경에서의 인증 오류 해결

### 🔧 기술적 개선

- 레시피 메타데이터 자동 분류 시스템 추가
- 식단 생성 알고리즘 성능 최적화
- 타입 안정성 향상 (MealSet 타입 도입)
